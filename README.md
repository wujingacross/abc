### 安装依赖

`npm install`

### 启动

`npm run watch`

### 执行 `npm link`

此时就可以使用 `eos` 命令了。

- `eos config get`
- `eos config set type directRegistry`
- `eos config set registry https://github.com/xxx/xxx.git`
- `eos init m myVue`

- `eos config set type users`
- `eos config set registry YvetteLau`
- `eos init vue-template myVue`

### 发布

开发完成后，即可发布至 npm.
