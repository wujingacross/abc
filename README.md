### 安装依赖

`npm install`

### 启动

`npm run watch`

### 执行 `npm link`

此时就可以使用 `wow` 命令了。

- `wow config get`
- `wow config set type directRegistry`
- `wow config set registry https://github.com/xxx/xxx.git`
- `wow init m myVue`

- `wow config set type users`
- `wow config set registry YvetteLau`
- `wow init vue-template myVue`

### 发布

开发完成后，即可发布至 npm.
